# 策略归档说明

归档时间: 2026-01-19 14:31:16

## 归档原因

经过30天实盘数据分析，V2和V3策略被证明无效：

- **V2策略** (limit_order_trend): 1,245笔交易，-$3,823，胜率48.8%
- **V3策略** (sustained_trend): 603笔交易，-$2,937，胜率34.2%
- **总计**: 30天亏损 -$6,641

核心问题：
1. EMA趋势跟踪滞后，只能入场在趋势尾部
2. 缺少多周期分析和位置评估
3. 没有"控制系统"进行综合决策

## 归档文件

- strategy_executor_v2.py - V2策略执行器
- contrarian_strategy.py - 反向操作策略（基于错误假设）
- futures_limit_order_executor.py - 限价单执行器

## 下一步

设计新的策略框架，包含：
1. 多周期分析模块（1D/4H/1H）
2. 位置评估模块（距离高低点、波动区间）
3. 决策仲裁模块（综合评分）
4. 只交易有正期望值的币种+方向组合
