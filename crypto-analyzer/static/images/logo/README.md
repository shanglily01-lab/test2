# AlphaFlow Logo 文件说明

## 文件列表

### 1. `alphaflow-logo-icon.svg`
**用途**：图标版本，用于favicon、应用图标
**尺寸**：200x200
**特点**：流动的Alpha符号，带三条数据流线条

### 2. `alphaflow-logo-full.svg`
**用途**：完整Logo（图标+文字），用于网站头部、品牌展示
**尺寸**：400x120
**特点**：左侧图标 + "AlphaFlow"文字 + 副标题

### 3. `alphaflow-logo-af.svg`
**用途**：字母组合版本，用于简洁场景
**尺寸**：200x200
**特点**：A和F字母通过数据流连接

### 4. `alphaflow-logo-cube.svg`
**用途**：3D立方体版本，代表多维度分析
**尺寸**：200x200
**特点**：3D立方体，带数据流动动画效果

### 5. `alphaflow-logo-minimal.svg`
**用途**：极简版本，用于小尺寸场景
**尺寸**：100x100
**特点**：最简化的Alpha符号

## 使用建议

- **网站Logo**：使用 `alphaflow-logo-full.svg`
- **Favicon**：使用 `alphaflow-logo-minimal.svg` 或 `alphaflow-logo-icon.svg`
- **应用图标**：使用 `alphaflow-logo-icon.svg`
- **品牌展示**：使用 `alphaflow-logo-full.svg`
- **简洁场景**：使用 `alphaflow-logo-af.svg` 或 `alphaflow-logo-minimal.svg`

## 颜色规范

- **主色**：#2B6FED (专业蓝)
- **辅色**：#7C3AED (科技紫)
- **渐变**：linear-gradient(135deg, #2B6FED 0%, #7C3AED 100%)

## 导出格式

如需其他格式，可以使用以下工具转换：
- **PNG**：使用 Inkscape、Figma 或在线工具
- **ICO**：使用 favicon.io 或 RealFaviconGenerator
- **不同尺寸**：16x16, 32x32, 64x64, 128x128, 256x256
