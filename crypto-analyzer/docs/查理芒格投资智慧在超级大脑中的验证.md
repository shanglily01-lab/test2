### 名言1："最重要的不是你摸到什么牌，而是能不能把手里的牌打好"

#### 芒格的原意

**核心思想**：
- 外部条件（摸到的牌）不可控
- 关键在于如何最大化利用现有资源（打好手里的牌）
- 这是一种**执行力哲学**：接受现实，专注于可控因素

**在投资中的应用**：
- 不要抱怨市场环境（牛市/熊市是"摸到的牌"）
- 关键是在任何市场环境下都能做出正确决策
- 不是等待"完美时机"，而是把握"当前机会"

#### 超级大脑的体现

### 名言3："投资需要大量的耐心而非大量的行动"

#### 芒格的原意

**核心思想**：
- 投资的本质是**等待**，而非频繁交易
- 大部分时间应该什么都不做（"耐心"）
- 只在极少数确定性高的机会出现时才行动
- 这是对"过度交易"的警告

**芒格的比喻**：
"我们的投资风格被称为'焦点投资'。我们寻找可以理解、有持续竞争优势、由我们欣赏和信任的人管理的企业。当找到这样的企业时，我们会大量买入并长期持有。"

**伯克希尔的实践**：
- 每年只投资极少数项目
- 持仓可达数十年（如可口可乐）
- 巴菲特："如果你一生只有20个投资机会，你会变成更好的投资者"

```python
# breakout_convergence.py
class BreakoutConvergence:
    """收敛机制 - 芒格"耐心而非行动"的算法实现"""

    def __init__(self, config: Dict = None):
        self.config = config or {
            # 🔥 核心：限制行动，强制耐心
            'max_duration': 4 * 3600,           # 4小时后失效（耐心等待下一次）
            'opening_window': 30 * 60,          # 30分钟窗口（集中行动）
            'max_openings': 5,                  # 每币种最多5次（不是无限行动）
            'reversal_threshold': 0.01,         # 1%反转失效（及时停止）
            'trend_end_threshold': 0.02         # 2%趋势结束失效（停止行动）
        }

        # 开仓记录（限制行动）
        self.max_openings_per_symbol = 5
        self.opened_symbols = {}  # {symbol: count}

    def can_open_more(self, symbol: str) -> dict:
        """检查是否还能开仓（强制耐心）"""

        current_count = self.opened_symbols.get(symbol, 0)

        if current_count >= self.max_openings_per_symbol:
            # 🔥 芒格式限制：已经行动5次，强制停止
            return {
                'can_open': False,
                'reason': f'{symbol}已开仓{current_count}次，达到上限（强制耐心）'
            }

        remaining = self.max_openings_per_symbol - current_count
        return {
            'can_open': True,
            'remaining': remaining,
            'message': f'{symbol}还可开仓{remaining}次'
        }

    def start_breakout_session(self, direction: str, strength: float):
        """开启破位会话 - 30分钟行动窗口"""

        self.current_session = {
            'session_id': f"{direction}_{int(time.time())}",
            'direction': direction,
            'strength': strength,
            'start_time': time.time(),
            'opening_window_end': time.time() + self.config['opening_window'],  # 30分钟
            'expiry_time': time.time() + self.config['max_duration'],  # 4小时
        }

        logger.info(
            f"[破位会话] {self.current_session['session_id']} 开始\n"
            f"  方向: {direction}\n"
            f"  强度: {strength:.1f}\n"
            f"  每个币种最多: {self.max_openings_per_symbol} 次开仓\n"
            f"  开仓窗口: {self.config['opening_window']/60:.0f} 分钟\n"
            f"  有效期: {self.config['max_duration']/3600:.1f} 小时"
        )

        # 🔥 芒格哲学：
        # - 30分钟内集中行动（"好机会要下重注"）
        # - 4小时后失效（"然后继续耐心等待"）
        # - 每币种最多5次（"不是大量行动"）
```

```python
def is_in_opening_window(self) -> bool:
    """检查是否还在30分钟开仓窗口"""

    if not self.current_session:
        return False

    current_time = time.time()
    window_end = self.current_session['opening_window_end']

    if current_time > window_end:
        # 🔥 窗口关闭：停止行动，恢复耐心

    return True

def is_session_expired(self) -> bool:
    """检查会话是否过期（4小时）"""

    if not self.current_session:
        return True

    current_time = time.time()
    expiry_time = self.current_session['expiry_time']

    if current_time > expiry_time:
        # 🔥 4小时后：这一轮机会结束，继续耐心等待下一轮
        logger.info(f"[收敛] 破位会话已过期（4小时），停止所有信号（恢复耐心）")
        return True

    return False
```

### 名言10："反向思考可帮助解决问题"（逆向思维）

#### 芒格的原意

**核心思想**：
- 不要只想"怎么成功"，更要想"怎么失败"
- 反向思考：避免失败比追求成功更重要
- "Invert, always invert"（反过来想，永远反过来想）

**芒格的方法**：
- 正向："我怎么在投资中赚钱？"
- 反向："我怎么在投资中亏钱？"→ 避免这些→ 赚钱

#### 超级大脑的体现

**✅ 逆向评分：顶部减分、底部加分**

```python

    # 芒格式反向：
    # - 不是"顶部能涨多少"（正向）
    # - 而是"顶部会亏多少"（反向）
    # - 避免失败比追求成功更重要
```

**反向思考：盈亏比**
