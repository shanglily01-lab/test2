- **链上数据提升 (15%)**: 大户钱包活动是最直接的信号

**关键变化**:
- ✅ **更大仓位**: 40%单笔仓位，抓住好机会重仓

### 5. 杠杆使用规则

**杠杆是双刃剑，激进但不鲁莽**

#### 杠杆倍数选择

```python
def select_leverage(signal_strength, market_volatility, consensus_level):
    """
    根据信号强度和市场状况选择杠杆
    """
    base_leverage = 3  # 基础杠杆

    # 信号强度调整
    if signal_strength > 70:
        base_leverage += 2
    elif signal_strength > 60:
        base_leverage += 1
    elif signal_strength < 50:
        base_leverage -= 1

    # 市场波动率调整
    if market_volatility > 0.08:  # 日波幅 > 8%
        base_leverage = min(base_leverage, 3)  # 高波动限制杠杆

    # 一致性调整
    if consensus_level == "STRONG_CONSENSUS":
        base_leverage += 1

    # 限制范围
    leverage = max(2, min(base_leverage, 8))

    return leverage
```

**杠杆使用建议**:
- 🔵 **2-3倍**: 日常使用，安全范围
- 🟢 **4-5倍**: 信号强烈时使用
- 🟡 **6-7倍**: 极强信号 + 趋势明确
- 🔴 **8倍**: 只在百年一遇的机会使用（如闪崩抄底）

**杠杆风险管理**:
```python
# 杠杆仓位的止损必须更严格
leverage_stop_loss = entry_price * (1 - 0.10 / leverage)

# 例如:
# 5倍杠杆 → 止损2% (避免爆仓)
# 10倍杠杆 → 止损1%
```

## 💡 激进投资者的核心心法

### 心法1: 快速决策

**机会稍纵即逝，不要犹豫**

```
看到信号 → 5分钟内决策 → 立即执行
```

**避免**:
- ❌ "我再看看"
- ❌ "等更好的价格"
- ❌ 过度分析导致错过机会

### 心法2: 敢于重仓

**好机会要重仓，小打小闹赚不了大钱**

```
弱信号 → 10-20%仓位 → 试探性交易
强信号 → 30-40%仓位 → 重仓出击
极强信号 → 40-50%仓位 → 全力以赴
```

**但要注意**:
- 总仓位不超过80%（留20%现金应急）
- 单币种不超过50%（分散风险）

### 心法3: 止损果断

**激进可以，但止损必须果断**

```
触及止损 → 立即平仓 → 不要犹豫
```

**避免**:
- ❌ "再给一点空间"
- ❌ "价格会反弹的"
- ❌ 把止损改成止盈

### 心法4: 让利润奔跑

**止损果断，止盈要贪婪**

```
盈利 > 20% → 不要急着全平
盈利 > 35% → 分批止盈
盈利 > 50% → 大部分止盈，剩余追踪
```

### 心法5: 顺势而为

**趋势是朋友，不要逆势**

```
强势上涨 → 做多，不做空
强势下跌 → 做空或观望
震荡市 → 减少交易
```

## ⚠️ 激进策略风险警告

### 风险1: 高回撤

**预期回撤: 20-40%**

**应对**:
- 心理准备：回撤是正常的
- 回撤 > 20%: 停止新开仓
- 回撤 > 30%: 暂停所有交易
- 回撤 > 40%: 立即清仓，重新评估策略

### 风险2: 连续亏损

**可能连续亏损5-8次**

**应对**:
- 连续亏损2次: 减少仓位50%
- 连续亏损3次: 暂停交易1天
- 连续亏损5次: 暂停交易1周，复盘策略

### 风险3: 杠杆爆仓

**高杠杆可能爆仓**

**预防**:
- 严格止损（杠杆止损更严格）
- 不在震荡市用高杠杆
- 保证金充足（至少50%余量）

### 风险4: 情绪失控

**连续盈利 → 过度自信**
**连续亏损 → 报复性交易**

**应对**:
- 设置日内最大亏损限制（如5%）
- 达到限制立即停止交易
- 每周休息1-2天

## 🎯 激进策略的适用时机

### 最适合的市场环境

✅ **强势单边行情** (牛市/熊市)
- 趋势明确，方向清晰
- 聪明钱大量涌入
- 建议：大胆重仓，让利润奔跑

✅ **极端恐慌/贪婪** (恐惧指数 < 20 或 > 80)
- 市场情绪极端，反转在即
- 建议：逆向操作，抄底或逃顶

✅ **重大事件驱动** (减半/ETF通过/监管政策)
- 基本面发生重大变化
- 建议：提前布局，事件前平仓

### 不适合的市场环境

❌ **震荡市** (价格上下反复)
- 假突破频繁，胜率降低
- 建议：减少交易频率，使用高频小单

❌ **低波动市** (日波幅 < 2%)
- 赚钱空间小，手续费吃掉利润
- 建议：观望，等待机会

❌ **不明方向** (技术指标混乱)
- 信号分歧严重
- 建议：不交易，宁可错过

## 💬 常见问题

### Q1: 激进策略会不会亏光？

**A**: 只要严格执行止损，不会亏光。最大回撤预期30-40%，但长期是盈利的。

### Q2: 需要盯盘吗？

**A**: 建议每天查看2-3次，或设置价格提醒。激进策略需要较快响应。

### Q3: 可以一直用激进策略吗？

**A**: 不建议。建议根据市场环境切换：
- 强趋势: 激进策略
- 震荡市: 保守策略或观望

### Q4: 杠杆一定要用吗？

**A**: 不一定。杠杆是放大器，初期可以不用杠杆，等熟练后再使用。

### Q5: 如何控制情绪？

**A**:
- 设置机械化规则
- 不要看短期盈亏
- 每周总结复盘
- 保持理性
