# Dashboard 使用指南

## 📊 概述

Dashboard（仪表盘）是加密货币智能分析系统的核心数据展示页面，整合了多维度数据分析，为投资决策提供全面的市场洞察。

## 🎯 核心功能

### 1. 多维度数据分析

Dashboard 整合了以下 5 个维度的数据：

#### 📈 技术指标分析
- **EMA 信号**：指数移动平均线交叉信号
- **RSI 指标**：相对强弱指标，判断超买超卖
- **MACD 指标**：移动平均收敛散度，趋势判断
- **成交量分析**：交易量变化趋势
- **价格趋势**：多周期价格走势

#### 📰 新闻情绪分析
- **实时新闻**：聚合多个新闻源的最新资讯
- **情绪标签**：利好/中性/利空自动分类
- **情绪评分**：量化情绪指标（0-100）
- **新闻来源**：显示发布媒体和时间
- **相关性分析**：新闻与价格波动的关联

#### 💰 资金费率监控
- **实时资金费率**：Binance 和 Gate.io 的资金费率
- **费率趋势**：历史费率变化
- **多空比**：合约市场多空持仓比例
- **持仓量变化**：合约持仓量趋势

#### 🧠 Hyperliquid 聪明钱追踪
- **大户持仓**：监控 Hyperliquid DEX 上的大户仓位
- **大额交易**：追踪大额买卖交易
- **PnL 分析**：盈亏表现分析
- **排行榜数据**：顶级交易者排名

#### ⛓️ 以太坊链上数据
- **链上指标**：活跃地址、交易量等
- **大额转账**：监控大额资金流动
- **交易所流入流出**：资金流向分析

### 2. 投资建议生成

基于 5 个维度的综合分析，系统自动生成投资建议：

- **强烈买入**：多个维度同时看涨，置信度高
- **买入**：整体偏向看涨
- **持有**：市场中性，建议观望
- **卖出**：整体偏向看跌
- **强烈卖出**：多个维度同时看跌，风险高

每个建议包含：
- **置信度**：0-100% 的置信度评分
- **理由说明**：详细的分析依据
- **风险提示**：潜在风险说明

### 3. 实时价格监控

- **当前价格**：实时市场价格
- **24小时涨跌**：价格变化百分比
- **价格图表**：可视化价格走势
- **多交易所对比**：Binance、Gate.io 等价格对比

### 4. 新闻卡片展示

- **新闻标题**：可点击跳转到原文
- **情绪标签**：直观显示利好/利空
- **发布时间**：相对时间显示（如"2小时前"）
- **来源媒体**：新闻发布平台
- **排序过滤**：按时间、情绪、相关性排序

### 5. 合约数据监控

- **永续合约价格**：实时合约价格
- **资金费率**：当前资金费率
- **持仓量**：合约持仓量数据
- **多空比**：市场多空比例

### 6. 系统统计

- **数据更新时间**：最后更新时间
- **监控币种数量**：当前监控的交易对
- **信号数量**：活跃交易信号
- **数据源状态**：各数据源连接状态

## 🎨 界面特性

### 现代化设计
- **深色主题**：专业的深色界面，保护眼睛
- **响应式布局**：适配不同屏幕尺寸
- **流畅动画**：平滑的过渡效果

## 🔄 数据更新机制

### 自动更新
- **前端刷新**：每 5 秒自动刷新数据
- **后端缓存**：使用缓存表提升性能
- **增量更新**：只更新变化的数据

### 手动更新
- **刷新按钮**：点击立即刷新所有数据
- **强制刷新**：清除缓存后重新加载

## 📊 数据来源

1. **价格数据**：Binance、Gate.io 等交易所 API
2. **新闻数据**：多个新闻源聚合
3. **技术指标**：基于历史 K 线数据计算
4. **资金费率**：Binance、Gate.io 合约数据
5. **Hyperliquid**：Hyperliquid DEX API
6. **链上数据**：以太坊区块链数据

## 🎯 使用场景

1. **市场概览**：快速了解整体市场状况
2. **投资决策**：基于多维度分析做出决策
3. **风险监控**：及时发现市场风险信号
4. **机会发现**：发现潜在投资机会
5. **趋势分析**：分析市场趋势和情绪

## 💡 使用技巧

1. **关注置信度**：优先关注高置信度的建议
2. **多维度验证**：结合多个维度数据判断
3. **及时响应**：关注实时新闻和信号
4. **风险控制**：注意风险提示
5. **历史对比**：对比历史数据判断趋势

## 🔧 技术实现

### 后端 API
- **路由**：`/api/dashboard`
- **缓存机制**：使用 `enhanced_dashboard_cached.py`
- **数据聚合**：并行获取多数据源
- **性能优化**：缓存表 + 异步处理

### 前端实现
- **框架**：原生 JavaScript + HTML5
- **图表**：Chart.js
- **样式**：CSS3 变量 + 现代设计
- **交互**：异步数据加载 + 实时更新

## 📝 注意事项

1. **数据延迟**：部分数据可能有 1-3 秒延迟
2. **网络依赖**：需要稳定的网络连接
3. **浏览器兼容**：建议使用现代浏览器（Chrome、Edge、Firefox）
4. **数据准确性**：仅供参考，不构成投资建议

## 🚀 快速开始

1. 打开浏览器访问 Dashboard 页面
2. 等待数据加载完成
3. 查看各维度数据分析
4. 根据投资建议做出决策
5. 定期刷新查看最新数据
