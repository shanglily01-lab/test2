## 🎯 不同场景的应对策略

### 场景1: 某币种连续失败
**识别**:
- 同一币种2次交易都亏损
- 例如: LIT/USDT (-12.44%, 如果再次亏损)

**行动**:
```sql
-- 从策略中移除该币种
UPDATE trading_strategies
SET config = JSON_REMOVE(config, '$symbols[x]')
WHERE id = 1768287501571;
```

或者手动编辑策略，从symbols列表中删除该币种。

### 场景2: 胜率突然下降
**识别**:
- 连续3笔有2笔亏损
- 或胜率从60%降到45%以下

**行动**:
1. 分析最近失败交易的notes字段
2. 检查是否都是同类平仓原因 (如: 5M金叉止损)
3. 考虑提高最小信号强度:
   ```python
   # 从1.0%提高到1.5%
   check_v3_sustained_trend() 中的
   MIN_SIGNAL_STRENGTH = 1.5
   ```

### 场景3: 市场进入震荡
**识别**:
- 限价单大量挂单但不成交
- 或成交后快速反转止损

**行动**:
1. 暂停策略1-2天
2. 等待明确趋势出现
3. 手动平仓浮亏持仓

### 场景4: 大幅盈利持仓
**识别**:
- 某持仓盈利超过+15%
- 例如: ALGO当前+10.65%

**行动**:
```
选项1: 手动止盈 (保守)
- 立即平仓锁定利润

选项2: 移动止盈 (激进)
- 等待自动移动止盈触发
- 可能吃到更大利润 (如NEAR +19.65%)
```
