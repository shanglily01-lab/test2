Hyperliquid 聪明钱指标说明
什么是聪明钱？
聪明钱是指在 Hyperliquid 平台上持续盈利、ROI 高的专业交易者。追踪他们的交易行为可以为投资决策提供重要参考。
核心指标
1. 监控钱包数

定义：系统当前监控的聪明钱钱包总数

筛选标准：

  • 累计盈亏 (PnL) > $10,000
  • 投资回报率 (ROI) > 20%
  • 账户价值 > $5,000

💡 意义：数量越多，统计越可靠。建议 1000+ 个钱包。

2. 24h交易量

定义：所有监控钱包在最近24小时的总交易金额（USD）

参考标准：

  • < $5M：市场较为平静
  • $5M - $20M：正常交易活跃度
  • $20M - $50M：市场活跃，有重要行情
  • > $50M：极度活跃，重大行情或事件

💡 意义：交易量增加表示市场活跃度上升，聪明钱正在积极布局。

3. Top活跃币种

定义：显示交易最活跃的前5个币种及其净流入/流出

计算公式：

净流入 = LONG交易总额 - SHORT交易总额

颜色含义：

  • 🟢 绿色 ▲：净流入为正，聪明钱偏向做多（看涨）
  • 🔴 红色 ▼：净流入为负，聪明钱偏向做空（看跌）

示例：

  • BTC ▲ $2.5M → 聪明钱在做多BTC，做多比做空多250万美元
  • ETH ▼ $1.8M → 聪明钱在做空ETH，做空比做多多180万美元

💡 强烈信号：净流入 > $2M 或 < -$2M 时，方向较为明确。

4. 最近大额交易

显示内容：按交易金额从大到小排序的前20笔交易

交易卡片信息：

  • 币种 + 方向： LONG 做多/看涨， SHORT 做空/看跌
  • 交易金额：该笔交易的名义价值（USD）
  • 钱包标签：交易者的标识，可追踪特定高手
  • 交易时间：成交时间
  • PnL (已实现盈亏)： 绿色 = 盈利， 红色 = 亏损

💡 关键：连续盈利的交易方向值得重点关注。

如何使用
📈 看涨信号组合
  • 24h交易量显著增加 (> $20M)
  • Top币种大多显示绿色 ▲（净流入为正）
  • 最近大额交易以 LONG 为主
  • 高盈利的 LONG 交易占比高
📉 看跌信号组合
  • Top币种大多显示红色 ▼（净流出为负）
  • 最近大额交易以 SHORT 为主
  • LONG 交易出现大额亏损
重要提示
  • 不是绝对信号：聪明钱也会犯错，需结合技术分析、基本面等多维度判断
  • 数据延迟：数据每30分钟更新一次，不适合极短线交易
  • 风险管理：即使跟随聪明钱，也要设置止损
  • 样本大小：监控钱包越多（建议 > 1000），数据越可靠

智能投资分析说明
概述

本系统采用 5维度智能分析模型，综合评估每个币种的投资价值， 为您提供数据驱动的投资建议。所有指标实时更新（每5秒刷新），确保您获得最新的市场洞察。

五大核心指标详解
📊 技术指标 (Technical Indicators)

分析内容：

  • RSI (相对强弱指数): 判断超买/超卖状态
    • RSI > 70 → 超买（可能回调）
    • RSI < 30 → 超卖（可能反弹）
  • MACD (平滑异同移动平均线): 捕捉趋势变化
    • 金叉 (MACD线上穿信号线) → 看涨
    • 死叉 (MACD线下穿信号线) → 看跌
  • 布林带 (Bollinger Bands): 识别价格区间
    • 价格触及下轨 → 超卖，可能反弹
    • 价格触及上轨 → 超买，可能回调
  • EMA (指数移动平均线): 判断趋势方向
    • 短期EMA > 长期EMA → 上升趋势
    • 短期EMA < 长期EMA → 下降趋势

评分规则：

  • 80-100分：多个指标强烈看涨/看跌，信号明确
  • 60-80分：部分指标支持，信号中等
  • 0-60分：指标冲突或信号弱，建议观望
📰 新闻情绪 (News Sentiment)

分析来源：

  • 新闻标题和内容： 来自 CoinDesk、Cointelegraph 等权威媒体
  • Reddit社区讨论： r/CryptoCurrency、r/Bitcoin 等热门社区
  • Twitter大V动态： 行业KOL的推文和观点
  • SEC监管公告： 美国证券交易委员会的官方消息

情绪分析方法：

  • NLP自然语言处理： 识别文本中的情绪倾向（积极/消极/中性）
  • 关键词权重： "突破"、"上涨" → 正面；"崩盘"、"下跌" → 负面
  • 时效性加权： 最近1小时的新闻权重最高，24小时前的权重递减

评分规则：

  • 80-100分：大量正面新闻，市场情绪积极
  • 40-60分：新闻中性，市场情绪平稳
  • 0-40分：负面新闻较多，市场情绪消极
💰 资金费率 (Funding Rate)

什么是资金费率？

资金费率是永续合约市场中，多头和空头之间定期支付的费用。 当资金费率为正时，多头支付给空头；为负时，空头支付给多头。

如何解读？

  • 资金费率 > 0.05%： 市场极度乐观，多头过热（可能回调风险）
  • 资金费率 0.01% ~ 0.05%： 市场看涨，多头占优
  • 资金费率 -0.01% ~ 0.01%： 市场中性，多空平衡
  • 资金费率 -0.05% ~ -0.01%： 市场看跌，空头占优
  • 资金费率 < -0.05%： 市场极度悲观，空头过热（可能反弹）

评分规则：

  • 80-100分：资金费率支持当前方向，趋势可靠
  • 60-80分：资金费率中性，无明确信号
  • 0-60分：资金费率与趋势背离，警惕反转
🧠 Hyperliquid聪明钱 (Smart Money on Hyperliquid)

什么是聪明钱？

聪明钱是指在Hyperliquid DEX上交易表现优异的钱包地址， 这些地址通过精准的交易决策积累了可观的利润（PnL）和高ROI。

监控标准：

  • PnL (已实现盈亏) > $10,000： 盈利能力强
  • ROI (投资回报率) > 20%： 回报率高
  • 账户价值 > $50,000： 资金量大，影响力强

分析维度：

  • 交易方向： 聪明钱正在做多(LONG)还是做空(SHORT)？
  • 交易金额： 单笔交易超过$50,000视为大额交易
  • 净流入/流出： 做多金额 - 做空金额 = 净流入
  • 交易时间： 最近24小时的活动最具参考价值

评分规则：

  • 80-100分：聪明钱大量买入/卖出，方向明确
  • 60-80分：聪明钱中等活跃，信号中等
  • 0-60分：聪明钱无明显动作或方向分散
⛓️ 链上数据 (On-Chain Data)

分析内容：

  • 大额转账监控： 追踪交易所和巨鲸地址的资金流动
    • 转入交易所 → 可能卖出（看跌）
    • 转出交易所 → 可能囤币（看涨）
  • 活跃地址数： 反映网络活跃度和用户兴趣
  • 交易量： 链上交易量增加表明市场热度提升
  • Gas费用： Ethereum网络Gas费飙升通常预示大量交易

监控地址：

  • Binance钱包： 最大的交易所，资金流动风向标
  • 巨鲸地址： 持币量巨大的个人或机构
  • DeFi协议： Uniswap、Aave等DeFi项目的智能合约

评分规则：

  • 80-100分：链上活动活跃，资金流向支持趋势
  • 60-80分：链上活动正常，无明确信号
  • 0-60分：链上活动低迷或资金异常流动
