关于超级大脑

超级大脑是一套专为 Binance 合约市场设计的全自动量化交易平台，基于 Python FastAPI 构建，7×24 小时不间断运行。系统通过 14 个独立信号组件对 550+ 交易对进行实时评分，融合 Big4 趋势过滤（BTC/ETH/BNB/SOL 加权）与 SmartExitOptimizer 风控，五条策略并行运行，自动识别优质进场机会并执行开平仓。

支持模拟盘与实盘双轨并行，实盘仅限 TOP50 高胜率交易对。所有策略参数可通过 Web 管理界面实时调整，无需重启服务。每日 00:00 UTC 自动生成复盘报告，分析信号组件胜率、优化权重，并通过 Telegram Bot 实时推送开平仓通知，全程透明可追溯。
